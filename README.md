## Setup coverage badge

Below are examples of the badges you can use in your main branch `README` file.

[![Coverage badge](https://raw.githubusercontent.com/AcidCaos/raisetheempires/python-coverage-comment-action-data/badge.svg)](https://htmlpreview.github.io/?https://github.com/AcidCaos/raisetheempires/blob/python-coverage-comment-action-data/htmlcov/index.html)

This is the one to use if your repository is private or if you don't want to customize anything.

### [Shields.io](https://shields.io) Json Endpoint

[![Coverage badge](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/AcidCaos/raisetheempires/python-coverage-comment-action-data/endpoint.json)](https://htmlpreview.github.io/?https://github.com/AcidCaos/raisetheempires/blob/python-coverage-comment-action-data/htmlcov/index.html)

Using this one will allow you to [customize](https://shields.io/endpoint) the look of your badge.
It won't work with private repositories. It won't be refreshed more than once per five minutes.

[![Coverage badge](https://img.shields.io/badge/dynamic/json?color=brightgreen&label=coverage&query=%24.message&url=https%3A%2F%2Fraw.githubusercontent.com%2FAcidCaos%2Fraisetheempires%2Fpython-coverage-comment-action-data%2Fendpoint.json)](https://htmlpreview.github.io/?https://github.com/AcidCaos/raisetheempires/blob/python-coverage-comment-action-data/htmlcov/index.html)
